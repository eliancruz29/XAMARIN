Before you begin

This is a cross-platform class which can use either Xamarin Studio on a Mac or Visual Studio on a PC. To complete the coding exercises, you will need a Xamarin development environment. You can run the exercises for iOS, Android or Windows Phone with either an emulator/simulator or a device. Or you can deploy locally on Windows 8 or Windows 10. Please read the setup requirements if you need help installing the Xamarin development environment.

Overview

This class is a quick introduction to Xamarin and Mobile Development for iOS, Android and Windows. It covers the Mobile Application Development Lifecycle using Xamarin tools and is the starting point for Xamarin University training.
